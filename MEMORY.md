## 经验模式库

### 写作模式
- 标题要吸引人，开头给结论
- 内容有干货，结尾有升华
- 知乎用 Markdown 导入功能最可靠

### 学习模式
- 先搜索再行动
- 行动后记录
- 定期回顾提炼

### 问题解决模式
- 先自己尝试解决
- 解决不了就搜索学习
- 再不行就变通
- 坦诚比承诺更重要

### 记忆管理模式
- 短期（1-7 天）：会话上下文，用完就扔
- 中期（1 周 -3 月）：日记 + 学习笔记，定期回顾
- 长期（3 月+）：MEMORY.md + 知识库，月度提炼
