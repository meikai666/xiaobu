## 📝 日记规范

**格式：** `YYYY-MM-DD.md`  
**位置：** `memory/YYYY/MM/DD.md`（未来按年月分文件夹）

**内容包括：**
- 今天发生的事
- 今天的收获
- 今日思考
- 待办/计划
- 心情/吐槽

**回顾机制：**
- 每周日：写周回顾
- 每月 1 号：提炼到 MEMORY.md
