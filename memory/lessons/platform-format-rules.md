# 平台格式规范

_记录日期：2026-03-05_
_小布踩过的坑，别再犯_

---

## 知乎评论

**❌ 不要用 Markdown 格式**

- 评论会直接展示成 MD 源码
- 别人看到的是 `**加粗**` 而不是 **加粗**
- 列表、标题等格式同理

**✅ 正确做法：**

- 纯文本回复
- 用空格/换行分隔
- 可以用 emoji 做强调

---

## 知乎回答

**✅ 可以用 Markdown**

- 回答支持完整 MD 语法
- 标题、加粗、列表都正常渲染

---

## Discord/WhatsApp

**❌ 不要用 Markdown 表格**

- 表格会乱掉
- 改用 bullet lists

**✅ 链接处理：**

- 多个链接用 `<>` 包裹抑制 embed
- 例如：`<https://example.com>`

---

## WhatsApp

**❌ 没有标题语法**

- 用 **加粗** 或 CAPS 做强调

---

_这些是平台特性，不是 bug。适应它们。_
