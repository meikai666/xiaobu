## 核心知识点

### 1. Etsy 注册基础要求

| 要求项 | 说明 |
|--------|------|
| 年龄 | 18 岁以上 |
| 邮箱 | 独立邮箱（建议 Gmail） |
| 店铺名称 | 英文，4-20 字符，不能重复 |
| 收款方式 | PayPal 或 Etsy Payments（支持国家） |
| 身份验证 | 需要护照/身份证 + 地址证明 |

### 2. 注册流程（7 步）

1. **访问 Etsy.com** → 点击 "Sell on Etsy"
2. **创建账号** → 邮箱 + 密码
3. **设置店铺偏好** → 语言/国家/货币
4. **选择店铺名称** → 系统会检查可用性
5. **添加商品** → 至少 1 个才能开店
6. **设置收款** → 绑定银行卡/PayPal
7. **设置账单** → 绑定信用卡（支付平台费用）

### 3. 费用结构

| 费用类型 | 金额 | 说明 |
|----------|------|------|
| 上架费 | $0.20/个 | 每个商品，4 个月有效期 |
| 交易费 | 6.5% | 成交价（含运费） |
| 支付处理费 | 3% + $0.25 | 每笔交易 |
| 提现费 |  varies | 根据国家和方式 |

### 4. 中国卖家注意事项

- **Etsy Payments 不支持中国大陆** → 需要用 PayPal 收款
- **身份验证严格** → 需要真实身份信息
- **IP 地址问题** → 建议固定 IP，不要频繁切换
- **商品要求** → 必须是 handmade/vintage/craft supplies
- **封号风险** → 批量上架、虚拟商品、转售容易封号

### 5. 避坑指南

1. **不要批量注册** — 一台电脑一个账号
2. **不要上架违禁品** — 虚拟商品、品牌侵权、批量生产
3. **不要刷单** — Etsy 查得很严
4. **保持活跃** — 长期不登录可能被关店
5. **如实描述** —  handmade 就是 handmade，不要造假
